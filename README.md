# ⚛️ Nuclear Weapon vs. Nuclear Reactor

An **interactive educational website** built to highlight and explain the key differences between a **nuclear weapon** and a **nuclear reactor** — in a clear, engaging, and visually appealing way.  

Developed with passion by **Ahmed Sameer**.

---

## 🚀 Built With

- [Next.js](https://nextjs.org) – React framework for production-grade web apps  
- [TailwindCSS](https://tailwindcss.com) – Modern utility-first CSS framework  
- [Framer Motion](https://www.framer.com/motion/) – Smooth and elegant animations  
- [next-i18next](https://github.com/i18next/next-i18next) – Seamless multilingual support  
- [next-themes](https://github.com/pacocoursey/next-themes) – Dark/light theme switching

## 🧠 About This Project

This project was created to **educate and correct common misconceptions** about nuclear technology.  
While both nuclear weapons and nuclear reactors rely on atomic energy, their goals, designs, and impacts are **completely different** — and this website demonstrates those contrasts interactively.
